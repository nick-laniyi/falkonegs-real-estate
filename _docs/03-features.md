# Features Overview

## Public Website

**Homepage**
- Hero section with search prompt
- Quick search form with filters
- Featured properties grid (6 properties)
- Features/benefits section
- Call-to-action section

**Property Listing Page**
- Filter sidebar (property type, transaction type, location, price)
- Sort options (newest, price low-high, price high-low)
- Grid/list view toggle
- Pagination
- Property cards with image, price, details, and favorite button

**Property Details Page**
- Main image gallery
- Property details (bedrooms, bathrooms, square feet)
- Full description
- Amenities list
- Location information
- Agent contact form
- Similar properties
- Save to favorites button

**User Authentication**
- Registration form with validation
- Login with remember me
- Password reset via email
- Session management

## User Dashboard (Logged-in Users)

**Dashboard Home**
- Welcome message
- Quick stats (saved properties, inquiries)
- Recent activity

**Saved Properties**
- List of favorite properties
- Remove from favorites
- View property details

**My Inquiries**
- History of property inquiries
- Status tracking (new, contacted, closed)
- View property details

**Profile Settings**
- Update personal information
- Change password
- Notification preferences

## Admin Panel

**Dashboard**
- Site statistics
- Recent inquiries
- Quick actions

**Property Management**
- Add new property
- Edit existing properties
- Delete properties
- Upload multiple images
- Set featured status
- Mark as sold/rented

**Inquiry Management**
- View all property inquiries
- Mark as read/contacted
- Reply to inquiries
- Delete spam

**Contact Messages**
- View contact form submissions
- Mark as replied
- Export to CSV

**User Management**
- View registered users
- Promote to agent/admin
- Disable accounts
